资本结构：
cap_struct_general_detail/agg，只计算一般工商业企业。这是通过限定balance_sheet表中comp_type=1实现的，该字段来自tushare的balance表。
现金：
	cash_plus：包括现金，其他流动资产（当前看到的主要是理财产品），投资性房地产，衍生金融资产。
	cash_minus包括预计负债。
	cash_all = cash_plus + 交易性金融资产 - cash_minus。
	note：cash_plus的概念范围有点过于广泛，但cash_all提供了一个特别综合的现金概念，有一定参考意义。
invest：
	交易性金融资产，可供出售金融资产，长期股权投资（equity investee）三项，当前是分开的。
wc+，wc-的归类原则是衡量业务运营过程的资金占用程度，其他的因素不要放进来。
预计负债：例如司法判决的债务。不应该算作WC-，因为这会造成WC_net偏低，仿佛是运营很高效，其实是吃官司导致的，从而导致分析南辕北辙。算作Liability也不合适，它在逻辑上和投入资本无关。调整为Cash-，从Cash中减去。
一般工商业企业中有金融业务的，把金融业务的条目剔除。	todo，还需要再考虑。
